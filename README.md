# basketball-game
APLIMAT Final Project

A final quiz/project for the class APLIMAT 2017 2nd term.  
Basketball game in OpenGL

## Controls
**Space** - Throw  
**W** - Increase Power  
**S** - Decrease Power  
**A** - Increase Angle  
**D** - Decrease Angle  
**F** - Show Aiming Line  
**R** - Reset Only  
**X** - Randomize and Reset  
**C** - Add Score
